# Overview

**od-do** — A Python object-based diagramming toolkit.

- **GitHub**: https://github.com/cart0113/od-do

## LLM Documentation

The user documentation in `/docs/src/user-guide/` and
`/docs/src/getting-started/` is written specifically so that an LLM can read
these markdown files and fully understand how to create diagrams with OD-DO.
